# --- Comandos en Bash ---
- **cd**   : Se utiliza para desplazarse entre carpetas
- **cd -** : Ingresa a la penultima carpeta
- **cd /home/user**: Ingresa carpetas por rutas
- **cd ..** : Retrocede 1 carpeta
